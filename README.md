# Loan Approval Prediction using Machine Learning

 Overview
This project predicts loan approval using Machine Learning. It includes EDA, data preprocessing, missing value handling, skewness treatment, feature scaling, and model training using Logistic Regression and Random Forest Classifier.

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

 Models Used
- Logistic Regression
- Random Forest Classifier

 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

 Dataset
Loan Prediction Dataset
